Running SASI Runner

This page describes how to run the SASI Runner program.
  1. Read the instructions.
  2. Select an input data folder:
    1. Click the 'select input' button.
    2. Navigate to the name of the input folder you wish to use, and click on the folder to highlight it.
    3. Click the 'Open' button.
    4. You should see "Selected '/path/to/your/file' as input." in the log pane.
  3. Specify an output file:
    1. Click the 'Specify output' button.
    2. Choose where you want to put the output file, and what you want to call it. Make sure the filename ends with '.zip' .
    3. Click the 'Save' button.
    4. You should see "Selected '/path/to/your/file' as output." in the log pane.
  4. Configure the model resolution by checking fields. If a field is a checked, you will be able to group results by that field. For example, if the 'energy' field is checked, you will get separate result rows for each energy level. Please note that increasing the resolution by enabling more fields can *greatly* increase run times and output file sizes.
  5. Run the runner:
    1. Click the 'Run' button. You should see log messages start to appear in the log pane.
    2. Wait for the runner to finish. This can take some time.
    3. When the runner is complete the progress bar will stop. The final message in the log pane should be something like 'SASI Run completed, output file is: '/path/to/the/output.zip'
