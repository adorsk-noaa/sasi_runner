Running SASI Gridder

This page describes how to run the SASI Gridder program.
  1. Read the instructions.
  2. Select an input data folder:
    1. Click the 'select input' button.
    2. Navigate to the name of the input folder you wish to use, and click on the folder to highlight it.
    3. Click the 'Open' button.
    4. You should see "Selected '/path/to/your/file' as input." in the log pane.
  3. Specify an output file:
    1. Click the 'Specify output' button.
    2. Choose where you want to put the output file, and what you want to call it. Make sure the filename ends with '.csv' .
    3. Click the 'Save' button.
    4. You should see "Selected '/path/to/your/file' as output." in the log pane.
  4. Run the gridder:
    1. Click the 'Run' button. You should see log messages start to appear in the log pane.
    2. Wait for the gridder to finish. This can take some time.
    3. When the gridder is complete the progress bar will stop. The final message in the log pane should be something like 'Gridding completed, output file is: '/path/to/the/output.csv'
